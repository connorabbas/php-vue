# PHP Mini Framework with Vue

Starter content for env.php file (create in /app directory):
```
<?php
// Enviroment specific variables

// Database
putenv("DB_HOST=127.0.0.1");
putenv("DB_USERNAME=root");
putenv("DB_PASSWORD=");
putenv("DB_NAME=");

// Site Base Directory
putenv("BASE_DIR=/php-mf/");
?>
```
